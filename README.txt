================================================================================
  DOCUFLOW - ONLINE DOCUMENT MANAGEMENT HTML TEMPLATE
================================================================================

Thank you for purchasing DocuFlow!

This is a modern, responsive HTML5 template designed for online document 
management platforms. It provides a complete frontend solution for websites 
offering PDF tools, file conversion, compression, and security services.

================================================================================
  QUICK START
================================================================================

1. EXTRACT FILES
   - Extract the downloaded ZIP file to your computer
   - You'll find all template files organized in folders

2. VIEW THE TEMPLATE
   - Open html/index.html in your web browser to view the homepage
   - Navigate to other pages through the menu

3. CUSTOMIZE
   - Open files in your preferred code editor
   - Edit HTML content directly
   - Modify colors in assets/css/style.css
   - Adjust functionality in assets/js/main.js

4. DEPLOY
   - Upload all files to your web server via FTP/SFTP
   - Ensure proper file permissions (644 for files, 755 for folders)
   - Test all pages after deployment

================================================================================
  CUSTOMIZATION TIPS
================================================================================

CHANGE COLORS:
Edit the CSS variables in assets/css/style.css:
:root {
    --primary-color: #4F46E5;  /* Change this value */
}

CHANGE FONTS:
1. Visit Google Fonts (https://fonts.google.com)
2. Select your font
3. Replace the font link in HTML files
4. Update --font-family in style.css

UPDATE LOGO:
Replace the icon-based logo in the navbar with your image:
<img src="../assets/images/your-logo.png" alt="Logo" height="40">

EDIT CONTENT:
Open any HTML file and edit the text directly. All content is clearly 
structured and commented.
