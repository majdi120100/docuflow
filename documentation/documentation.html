1. Introduction

Thank you for purchasing DocuFlow! This documentation will help you understand how to set up and customize your new template.

What is DocuFlow?

DocuFlow is a modern, responsive HTML5 template designed for online document management platforms. It provides a complete frontend solution for websites offering PDF tools, file conversion, compression, and security services.

Template Features

  • 6 fully responsive HTML pages
  • Built with Bootstrap 5
  • Dark mode support
  • Modern, clean design
  • Easy to customize
  • Cross-browser compatible
  • SEO optimized
  • W3C validated HTML5 & CSS3
  • Well-commented code
  • Comprehensive documentation

2. Requirements

Technical Requirements

To use this template, you need:

  • A modern web browser (Chrome, Firefox, Safari, Edge)
  • A code editor (VS Code, Sublime Text, Atom, etc.) - for customization
  • Basic knowledge of HTML and CSS - for customization
  • A web server or hosting (for production deployment)

Browser Compatibility

DocuFlow is compatible with the following browsers:

  • Chrome (latest 2 versions)
  • Firefox (latest 2 versions)
  • Safari (latest 2 versions)
  • Edge (latest 2 versions)
  • Opera (latest 2 versions)
Note: Internet Explorer is not supported as it has been discontinued by Microsoft.

3. File Structure

Understanding the file structure will help you navigate and customize the template:

docuflow/ ├── html/ │ ├── index.html │ ├── tools.html │ ├── tool-single.html │ ├── about.html │ ├── contact.html │ └── 404.html ├── assets/ │ ├── css/ │ │ └── style.css │ ├── js/ │ │ └── main.js │ ├── images/ │ │ └── placeholders/ │ └── fonts/ ├── documentation/ │ └── documentation.html └── README.txt

File Descriptions

  • tools.html - Complete listing of all available tools with search and filter functionality
  • tool-single.html - Template for individual tool pages with upload, process, and download workflow

  • 404.html - Custom error page for broken links
  • style.css - All CSS styles including responsive design and dark mode
  • main.js - All JavaScript functionality including theme toggle, file upload, and form validation

4. Installation

Local Installation (for development)

  1. Extract the downloaded zip file to your computer
  2. Navigate to the extracted folder
  3. Open html/index.html in your web browser
  4. You can now view and test the template locally
Important: Some features like form submission will not work properly when viewing files directly from your computer. For full functionality testing, use a local web server.

Using a Local Web Server

For better testing, use a local web server:

Option 1: VS Code Live Server

  1. Install Visual Studio Code
  2. Install the "Live Server" extension
  3. Open the template folder in VS Code
  4. Right-click on index.html and select "Open with Live Server"

Option 2: Python Simple Server

# Navigate to the template directory in terminal
cd path/to/docuflow

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Open browser to http://localhost:8000

Server Installation (for production)

  1. Upload all files to your web server via FTP/SFTP
  2. Ensure file permissions are set correctly (typically 644 for files, 755 for folders)
  3. Access your website through your domain name
  4. Test all pages and functionality

Recommended Hosting Providers

  • Any hosting provider that supports HTML/CSS/JavaScript
  • No special server requirements needed
  • CDN integration recommended for Bootstrap and icons

5. Customization

Changing Colors

Colors are defined using CSS variables in style.css. To change the color scheme:

/* Find this section in style.css */
:root {
    --primary-color: #4F46E5;        /* Main brand color */
    --primary-hover: #4338CA;        /* Hover state */
    --primary-light: #EEF2FF;        /* Light variant */
    
    /* Change these values to your preferred colors */
}

Changing Fonts

The template uses Google Fonts (Inter). To change the font:

  1. Visit Google Fonts
  2. Select your preferred font
  3. Copy the embed code
  4. Replace the font link in the HTML <head> section
  5. Update the CSS variable in style.css:
:root {
    --font-family: 'Your Font Name', sans-serif;
}

Updating Logo

The template uses an icon-based logo. To add a custom logo:

  1. Prepare your logo image (PNG or SVG format recommended)
  2. Save it in assets/images/
  3. Find this code in all HTML files:
<a class="navbar-brand" href="index.html">
    <i class="bi bi-file-earmark-text-fill text-primary fs-3"></i>
    <span class="fw-bold ms-2">DocuFlow</span>
</a>

Replace with:

<a class="navbar-brand" href="index.html">
    <img src="../assets/images/your-logo.png" alt="Your Logo" height="40">
</a>

Editing Content

All content is in plain HTML and can be edited directly:

  1. Open the HTML file you want to edit
  2. Find the section you want to change
  3. Edit the text between the HTML tags
  4. Save the file
  5. Refresh your browser to see changes

Adding/Removing Tools

To add a new tool to tools.html:

<div class="col-md-6 col-lg-4 tool-item" data-category="optimize" data-name="Your Tool Name">
    <div class="tool-card">
        <div class="tool-icon">
            <i class="bi bi-your-icon"></i>
        </div>
        <h5 class="fw-semibold mb-2">Your Tool Name</h5>
        <p class="text-muted small mb-3">Tool description</p>
        <a href="tool-single.html" class="btn btn-outline-primary btn-sm">
            Use Tool
            <i class="bi bi-arrow-right ms-1"></i>
        </a>
    </div>
</div>

Customizing Dark Mode

Dark mode colors are defined in the [data-theme="dark"] section of style.css:

[data-theme="dark"] {
    --primary-color: #6366F1;
    --white: #111827;
    --black: #F9FAFB;
    /* Adjust these as needed */
}

Modifying Animations

Animation speeds and effects can be adjusted in the CSS variables:

:root {
    --transition-fast: 150ms;
    --transition-base: 300ms;
    --transition-slow: 500ms;
}
